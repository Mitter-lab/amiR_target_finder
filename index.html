amiR-target-finder


amiR-target-finder identifies a minimum set of artificial micoRNA/s (and their star strands) that can target all sequences in a FASTA format reference sequence file.

Developed by Stephen Fletcher @ the laboratory of Assoc. Prof. Neena Mitter, University of Queensland

Usage:


amiR_target_finder input_file [-min_win] [-max_win]


Positional arguments:

input_file : FASTA file containing reference sequences


Optional arguments:

-h : show this help message and exit
-min_win : Min. window size to iterate from
-max_win : Max. window size to iterate to
-V : show program's version number and exit


Output:


Example:

amiR_target_finder ref_file.fa -min_win 21 -max_win 22

--------------------------------------------------
44 reference sequences loaded
win = 21
no of amiRs = 2

Recommended amiRNAs/amiRNA*s are:

1. ('TCTTCACAAAAAGATAATAAG', 'CTTATTATCTTTTTGTGAAGA')
2. ('TTCCAGGGTTGTTTTTACTCA', 'TGAGTAAAAACAACCCTGGAA')

Test all seqs targeted = True
